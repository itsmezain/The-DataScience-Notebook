1- Why use SQL for Data Cleaning in real world production scenirio when we already have pandas and numpy?

3- There are 3 datatypes in SQL:
	i) Numerical : int, deci
	ii) Text: var
	iii) Temporal: date, time, etc
4- String Data Types:
	i) CHAR
	ii) VARCHAR
	iii) TEXT
	iv) MEDIUMTEXT
	v) LONGTEXT

5- Wildcards operator: % and _
